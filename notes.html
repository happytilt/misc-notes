Variables & Data Types

  • Variables store data values.
  • Data Types:
    • int (integers)
    • float (decimal numbers)
    • str (strings/text)
    • bool (True/False)
    • NoneType (None)
  • print(type(var)) - prints out data type of var
  • Example:
    name = "Alice"
age = 25
height = 5.9
is_student = True

Numbers

  • int: whole numbers, e.g., 5
  • float: decimal numbers, e.g., 5.5
  • Math operations:
    +, -, *, /, % (modulus) // (floor division), ** (exponent)

Booleans & Operators

  • Boolean: True or False
  • Comparison operators: ==, !=, <, >, <=, >=
  • Logical operators: and, or, not

= (single equals) vs == (double equals)

  • Single Equals
    • Assignment operator, not for comparison
  • Double Equals
    • Equality comparison operator, meant for comparison and returns True/False

String Formatting

  • Inserting variables into strings:
    name = "Bob"
print(f"Hello, {name}")
print("Hello, {} {}".format(first, last))
  • Escape characters you can use inside strings
    • \n - new line
    • \t - tab indent
    • \\ - backslash (\)
    • \' or \" - single/double quote
    • \r - Carriage return (move to beginning of line)
  • {} placeholders
    • .format() is a string method used to insert variables (or values) into a string.
  • String prefixes change how the string behaves when it is created
    • r (Raw string) - Keeps backslashes (\) literally
    • f (Formatted string) - Better than .format()
    • b (Byte string) - Stores binary data and not text
    name = "Alice"
print(f"Hello, {name}")  # f-string: inserts variable
print(r"C:\Users\Alice")  # r-string: keeps \\
print(b"Hello")           # b-string: bytes, not text

Data Structures

  • Tuple (()): A tuple is immutable, meaning once you create it, you can't change its elements. It's great for storing fixed data—like coordinates, RGB color values, or database records.
  • List ([]): A list is ordered and mutable, making it ideal for situations where you need to add, remove, or modify items dynamically—such as managing tasks in a to-do list or storing user input.
  • Set ({}): A set is unordered and contains unique elements only, meaning duplicates are automatically removed. It's useful for operations like finding unique values, checking membership quickly, or performing mathematical set operations (union, intersection).
  • Dictionary ({key: value}): A dictionary allows you to store key-value pairs, making it ideal when you need to map one piece of data to another—like storing user profiles ({"name": "Alice", "age": 25}) or caching search results.

Tuples

  • Immutable, iterable ordered collection.
  • Defined with parentheses ().

Example:

my_tuple = (1, 2, 3)

Lists

  • Mutable, iterable ordered collection.
  • Defined with brackets [].
  • Methods
    • append(item) – Adds an item to the end of the list.
    • extend(iterable) – Extends the list by appending all elements from an iterable.
    • insert(index, item) – Inserts an item at a specific index.
    • remove(item) – Removes the first occurrence of the specified item.
    • pop(index) – Removes and returns the item at the specified index (or the last item if no index is given).
    • index(item) – Returns the index of the first occurrence of the item.
    • count(item) – Returns the number of occurrences of the item.
    • sort() – Sorts the list in ascending order (modifies the list in place).
    • reverse() – Reverses the order of the list.
    • copy() – Returns a shallow copy of the list.
    • clear() – Removes all items from the list.

Example:

my_list = [1, 2, 3]
my_list.append(4)

Sets

  • Mutable, iterable and unordered collection of unique elements.
  • Defined with {} or set().
  • Example:
    my_set = {1, 2, 3, 3}  # {1, 2, 3}

Dictionaries

  • Key-value pairs.
  • Defined with curly braces {}.
  • Example:
    my_dict = {"name": "Alice", "age": 25}
print(my_dict["name"])

Conditionals

  • Decision making with if, elif, else.
  • Once one branch runs, the rest are ignored.
    if x > 0:
    print("Positive")
elif x == 0:
    print("Zero")
else:
    print("Negative")
  • elif vs else
    • elif (short for “else if”) - can have as many as needed
    • else - comes last; a catch all case

Loops

  • For loop: iterate over a sequence.
  • While loop: repeat while condition is True.
  • Examples:
    # Loop through a list
fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(f"I love {fruit}!")

# Prints 0, 1, 2, 3, 4
for i in range(5):
    print(f"Number: {i}")

# A countdown using while loop
count = 5
while count > 0:
    print(f"Counting down: {count}")
    count -= 1  # Reduce count by 1 each time
  • Control Statements
    • break: Exits the loop early.
    • continue: Skips the current iteration.
    • pass: Acts as a placeholder (does nothing).

Reading and Writing Files

  • Open files with open().
  • Example:
    with open('file.txt', 'r') as textfile:
    content = textfile.read()

with open('file.txt', 'w') as textfile:
    textfile.write("Hello World")
  • With modes
    • r (Read mode) – Opens a file for reading. Raises an error if the file doesn’t exist.
    • w (Write mode) – Opens a file for writing. Creates a new file if it doesn’t exist, but overwrites existing content.
    • a (Append mode) – Opens a file for writing, but does not overwrite existing content. Adds new content at the end.
    • x (Exclusive creation mode) – Creates a new file and opens it for writing. Fails if the file already exists.
  • With statement
    • ensures automatic file closure, reducing potential errors
    with open("example.txt", "r") as file:
    content = file.read()
    print(content)  # File automatically closes after this block
  • File Input Output Methods
    • read(size) - Read up to size bytes/chars. If omitted or negative, reads the rest of the file into one string
    • write(string) - Writes string into file
    • seek(offset, whence) - Move the file’s “cursor” to byte position offset.
      • whence lets you choose start/end/current
      • 0 - Start
      • 1 - Current
      • 2 - End
    • close() - Close the file. (When you use with, it’s called automatically at block exit)

User Input

  • input() - Get input from the user:
    name = input("Enter your name: ")
age = int(input("Enter your age: "))

Exceptions and Error Handling

  • Catch errors with try, except block
    try:
    data = json.loads(text)
except (ValueError, TypeError):
    print("Bad or missing data!")

  • else clause - Runs only if no exception
    try:
    f = open(path)
except FileNotFoundError:
    print("No such file")
else:
    contents = f.read()
    f.close()
  • finally clause - Always runs, whether an exception occurred or not
    try:
    conn = open_connection()
    do_work(conn)
except NetworkError as e:
    log(e)
finally:
    conn.close()
  • raise clause - Signal an error yourself
    def withdraw(balance, amount):
    if amount > balance:
        raise ValueError("Insufficient funds")
    return balance - amount

Comprehensions

  • Short way to create lists, sets, or dicts.
  • Examples:
    squares = [x**2 for x in range(5)]
even_numbers = {x for x in range(10) if x % 2 == 0}
name_lengths = {name: len(name) for name in ["Alice", "Bob"]}

Functions and Code Reuse

  • Define functions with def.
  • Example:
    def greet(name):
    return (f"Hello, {name}")

print(greet("Alice"))

Lambdas

  • Anonymous (one-line) functions.
  • Syntax: lambda arguments: expression
  • Example:
    add = lambda x, y: x + y
print(add(2, 3))  # 5
